# go-gin-framework
一个简单的gin使用示例，可通过配置文件开启/关闭gin-swagger

##### 如何运行
`make help`

##### 如何生成文档
1. 安装swag：[安装方法](https://github.com/swaggo/swag)
2. 使用特定格式添加接口函数注释，注释格式参考[go-gin-example](https://github.com/eddycjy/go-gin-example)
3. swag init

##### 接口示例
```bash 
# 启动服务后访问以下地址
http://127.0.0.1:8074/swagger/index.html
```
